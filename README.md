# fishka.bio

Free, browser-based bioinformatics tools for sequencing workflows.

[Open fishka.bio](https://fishka.bio)

## Tools

- **ab1lens** — view and compare `.ab1` Sanger chromatograms in the browser.
- **seqio** — sequencing file parsers and writers for bioinformatics data.
- **seqalgo** — sequence algorithms for alignment and analysis.

## Why browser-based?

- No installs.
- Fast local workflows.
- Private by default.
- Useful for quick inspection, comparison, and quality checks.
